FES

Full Express Server (FES) is a full express server setup and run script :)

It allows you to fully setup an http express-based web server that can give you pages and also subdomains!
All you have to do is download FESStart.js and run "node FESStart.js" and the rest is handled for you.

This is the home example, you can delete this file and replace it with your own, just make sure to look in "/src/ExpressServerSettings/config.json" for the config file and make any needed changes.

Check the Repository for more info.
